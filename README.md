# Color Space Interpolation

A color interpolation tool built with React, TypeScript, and Vite. Pick start and end colors, choose a color space, and view interpolated steps in real time.

## Stack

- React 19 + TypeScript
- Vite 7
- Tailwind CSS v4

## Commands

- `npm run dev` – Start dev server
- `npm run build` – Production build
- `npm run lint` – Run ESLint
- `npm run knip` – Find unused code and dependencies
- `npm run preview` – Preview production build
